2 · Python-koden som kjørerkjøres lokalt
def beregn(lengde, bredde, tykkelse, tetthet):
    volum = lengde * bredde * tykkelse   # m3
    masse = volum * tetthet              # tonn
    return {"volum": volum, "masse": masse}
Slik virker det · Pyodide laster en komplett Python-kjøretid (WebAssembly) inn i nettleseren. Beregningen over kjøres som ekte Python — alt skjer lokalt på maskinen, uten server og uten at data forlater nettleseren.
Mal for nye verktøy · Bytt ut funksjonen beregn() med din egen beregning, koble inndatafeltene til den, og legg verktøyet inn som et nytt kort på verktøysiden.
